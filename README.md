# 📊 Used Bike Price Analysis – Exploratory Data Analysis (EDA)

## 🚀 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a dataset of used bikes to identify the key factors influencing **bike resale prices**.  
The analysis explores how variables such as **bike age, kilometers driven, engine capacity (CC), and power** affect the price of used bikes.

---

## 🎯 Business Objective
To analyze used bike data and uncover **key insights that help buyers, sellers, and dealers understand the factors influencing resale prices**.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Microsoft PowerPoint
- ChatGPT

---

## 📂 Dataset Overview
The dataset contains information about used bikes, including:

- Price  
- Model Year  
- Kilometers Driven  
- Engine Capacity (CC)  
- Power (BHP)  
- Brand  

---

## 🧹 Data Cleaning
- Handled missing values
- Checked and removed duplicates
- Corrected data types for analysis

---

## ⚙️ Feature Engineering
- Created **Bike Age** from model year
- Extracted **Brand** from bike name
- Created **Age Groups** for better analysis

---

## 📊 Data Analysis & Visualizations
The following visualizations were used for analysis:

- Price Distribution
- Price vs Kilometers Driven
- Price vs Model Year
- Pairplot of Key Features
- Correlation Heatmap

---

## 🔍 Key Insights
- Bike prices **decrease with increasing age and kilometers driven**.
- Bikes with **higher engine capacity (CC) and power** tend to have higher resale prices.
- Most bikes fall within the **affordable to mid-range price category**.

---

## 💡 Recommendations
- Consider **bike age, kilometers driven, and engine performance** when pricing used bikes.
- Lower usage bikes tend to retain **higher resale value**.
- Categorizing bikes based on **age and performance** can improve pricing strategies.

---

## 📈 Project Outcome
This project demonstrates how **Exploratory Data Analysis (EDA)** can transform raw data into **meaningful insights that support better decision-making in the used bike market**.
