# Used Bike Price Analysis – Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a used bike dataset to understand the key factors affecting bike resale prices.

## Objectives
- Analyze the relationship between bike price and features such as:
  - Bike Age
  - Kilometers Driven
  - Engine Capacity (CC)
  - Power (BHP)
- Identify trends and correlations using data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- PowerPoint

## Key Insights
- Bike prices decrease with increasing age and kilometers driven.
- Bikes with higher CC and power tend to have higher resale prices.
- Most bikes fall within the affordable price range.

## Visualizations
- Price Distribution
- Price vs Kilometers Driven
- Price vs Model Year
- Pairplot of Key Features
- Correlation Heatmap
